## BuhoCleaner 是什么？

BuhoCleaner 是 Dr.Buho 出品的一款专为 macOS 打造的磁盘清理与系统优化工具，支持 Apple M1-M5 全系列芯片，兼容最新的 macOS Tahoe，中文界面，国人用起来没有障碍。

它的核心定位很简单：**让 Mac 重新变快，把空间还给你**。

功能涵盖：
- **垃圾清理（Flash Clean）**：一键扫描系统缓存、日志、DMG 安装包等，几分钟清出几 GB 甚至几十 GB 空间
- **应用深度卸载**：彻底清除 App 本体和残留文件，告别"删了 App 还占空间"的困扰
- **大文件 / 重复文件扫描**：找出那些你忘了存在的大文件，以及成堆复制粘贴出来的重复文件
- **启动项管理**：禁用开机自启的无用服务，让开机速度肉眼可见地提升
- **系统状态监控（菜单栏）**：实时显示 CPU 负载、内存占用、风扇转速、网速，一眼了解 Mac 状况
- **实用工具包**：Xcode 垃圾清理、碎纸机（永久删除文件）、内存释放、刷新 DNS 缓存、重建 Spotlight 索引……

一句话概括：**你以为 Mac 不需要清理，其实只是你没用对工具**。
